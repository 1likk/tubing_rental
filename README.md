# Система управления прокатом тюбингов

Веб-приложение для автоматизации работы проката тюбингов с возможностью управления арендой, отслеживания инвентаря и ведения финансового учета.

## Описание

Система предназначена для упрощения работы проката тюбингов. Приложение позволяет отслеживать состояние каждого тюбинга в режиме реального времени, управлять сессиями аренды, автоматически рассчитывать стоимость и формировать финансовые отчеты.

## Основные возможности

### Управление арендой
- Запуск и завершение сессий аренды
- Таймеры в реальном времени для каждой активной аренды
- Автоматический расчет стоимости (1000₸ за первый час, далее 0.28₸/сек)
- Визуальное отображение статуса каждого тюбинга
- Компактная сетка для управления большим количеством инвентаря (60+ единиц)

### Инвентарь
- Отслеживание статуса тюбингов (доступен, занят, в ремонте)
- Удобная карточная визуализация
- Цветовая индикация статуса

### Финансовая аналитика
- Дашборд с ключевыми показателями
- График динамики доходов
- Статистика за день и месяц
- История завершенных аренд

## Использование

### Добавление тюбингов
1. Войдите в админ-панель: `http://127.0.0.1:8000/admin`
2. Перейдите в раздел "Тюбинги"
3. Добавьте новый тюбинг с уникальным номером

### Управление арендой
1. Откройте раздел "Аренда"
2. Выберите доступный тюбинг
3. Нажмите "Старт" и введите имя гостя
4. Для завершения нажмите "Завершить" - система автоматически рассчитает стоимость

### Просмотр финансов
1. Откройте раздел "Финансы"
2. Просмотрите статистику и графики
3. Данные обновляются автоматически

## Модели данных

### Tubing (Тюбинг)
- `number` - уникальный номер тюбинга
- `status` - статус (available, busy, broken)
- `created_at` - дата создания

### RentalSession (Сессия аренды)
- `tubing` - связь с тюбингом
- `guest_name` - имя гостя
- `start_time` - время начала
- `end_time` - время окончания
- `status` - статус (active, completed)
- `final_cost` - итоговая сумма

## Расчет стоимости
Система использует следующую схему ценообразования:
- Первый час: 1000₸ (фиксированная ставка)
- После первого часа: 0.28₸ за секунду
